# Simple PHP Formmailer
##Requirements
* PHP 5.5+

## DEPENDENCIES
* Twitter Bootstrap for production
* PHPUnit, Composer: see [composer dependency manager](https://getcomposer.org/) for installation notes

## FEATURES are
* PHP Mailer with attachment
* Serverside Validation
* Fileupload
* Twitter Bootstrap
* CAPTCHA (optional)
* CSRF Protection
